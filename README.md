# AgentDeck

MPK Mini Mk4 as a Claude Code / opencode status deck. See [CLAUDE.md](CLAUDE.md) for the full
project brief and build plan.

## Setup

```bash
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
```

## Running

```bash
.venv/bin/python -m daemon.main
```
